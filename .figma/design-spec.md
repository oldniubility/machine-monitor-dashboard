# Figma 设计规范 — 机床数据监控看板

## 全局样式

### 颜色
| Token | 色值 | 用途 |
|-------|------|------|
| surface | #0F172A | 页面背景 (深色) |
| surface-alt | #1E293B | 卡片/容器背景 |
| border | #334155 | 分割线/边框 |
| accent | #3B82F6 | 主色调/链接/选中 |
| success | #22C55E | 运行中/在线 |
| warning | #EAB308 | 待机 |
| danger | #EF4444 | 故障/报警 |
| muted | #94A3B8 | 次要文字 |
| text | #F1F5F9 | 主要文字 |

### 字体
- 中文: Inter / PingFang SC / Microsoft YaHei
- 字号: 标题 24px, 副标题 14px, 正文 13px, 辅助 12px

### 间距
- 页面 padding: 24px
- 卡片间距: 16px (gap-4)
- 图标大小: 20-24px

## 页面结构

### 1. 顶部导航栏 (56px)
```
[Logo 图标] [机床监控看板]  [车间总览] [统计报表] [报警记录] [系统配置]     [🔔] [👤 管]
```
- 高度: 56px
- 背景: surface-alt
- 底部边框: border 1px
- 导航项: 图标+文字, 激活态 bg-accent/15 text-accent
- 右侧: 通知图标 + 用户头像(圆形, 32px)

### 2. 车间总览看板 (Dashboard)

#### 2.1 页面头部
```
🖥 车间总览              📡 实时连接 | 2026年6月3日
实时监控所有机床设备运行状态
```
- 标题 24px font-semibold
- 右侧: 连接状态指示 + 日期

#### 2.2 KPI 卡片行 (4列网格)
```
┌─────────────┐ ┌─────────────┐ ┌─────────────┐ ┌─────────────┐
│ 🖥           │ │ 📦          │ │ 📈           │ │ ⚠           │
│ 在线机床      │ │ 今日总产量    │ │ 整体稼动率    │ │ 活跃报警      │
│ 3 / 3       │ │ 2,799       │ │ 66.7%       │ │ 18          │
│ 启用的设备数   │ │ 件           │ │ 今日          │ │ 条未处理      │
└─────────────┘ └─────────────┘ └─────────────┘ └─────────────┘
```
- 卡片: surface-alt bg, border, rounded-lg, p-4
- 左侧图标 24px, 右侧文字
- 标题 12px uppercase text-muted
- 数值 20px font-semibold
- 副标题 12px text-muted

#### 2.3 筛选栏
```
筛选: [全部] [🟢运行中] [🟡待机] [🔴故障] [⚫离线]
```
- 按钮: px-3 py-1 rounded text-xs
- 激活: bg-accent text-white
- 默认: bg-surface-alt border text-muted

#### 2.4 设备列表表格
```
┌────────┬──────────┬────────────────┬─────────────┬────────┬──────────┬──────────┬────────┐
│ 状态   │ 设备编号  │ 设备名称        │ 品牌/型号    │ 车间   │ 今日产量  │ 运行时长  │ 报警   │
├────────┼──────────┼────────────────┼─────────────┼────────┼──────────┼──────────┼────────┤
│ 🟢运行中│ D001     │ CNC-品牌A-001  │ BrandA/A100 │ 一车间 │ 1,234   │ 8.5h    │ —     │
│ 🟡待机  │ D002     │ CNC-品牌B-002  │ BrandB/B200 │ 一车间 │ 890     │ 6.2h    │ —     │
│ 🔴故障  │ D003     │ CNC-品牌C-003  │ BrandC/C300 │ 二车间 │ 675     │ 4.1h    │ ⚠超温  │
└────────┴──────────┴────────────────┴─────────────┴────────┴──────────┴──────────┴────────┘
```
- 表头: border-b, text-muted, text-left/right
- 行: hover:bg-white/5, cursor-pointer
- 状态指示灯: 圆点+文字, rounded-full pill
- 编号: font-mono text-xs
- 产量: font-mono tabular-nums text-right

# 使用手册

## 登录

启动后访问看板，首次看到登录页：

- **管理员**：`admin` / `admin123`（可管理设备、模板、用户）
- **操作员**：需由管理员在「系统配置 → 用户管理」创建，可确认/解决报警
- **查看者**：只读访问

---

## 功能导航

顶部四个导航 Tab：

### 1. 车间总览（首页）

| 区域 | 说明 |
|------|------|
| KPI 卡片 | 在线机床数、今日总产量、整体稼动率、活跃报警数 |
| 状态筛选 | 按运行/待机/故障/离线筛选设备列表 |
| 设备列表 | 表格展示所有机床，显示实时状态、产量、报警 |
| 连接状态 | 右上角显示 WebSocket 实时连接或轮询模式 |

**操作**：点击设备行 → 进入单机详情页。

### 2. 统计报表

- **日期筛选**：选择起止日期分析历史数据
- **车间汇总卡片**：各车间在线设备数、产量、平均稼动率
- **产量对比图**：多设备产量折线图，hover 查看详情
- **稼动率统计表**：每台设备的运行/待机/故障/离线时长
- **导出 CSV**：点击「导出 CSV」下载产量明细

### 3. 报警记录

- **筛选**：按状态（未解决/未确认/已确认/已解决）、类型（故障/离线/通信异常）、设备
- **操作按钮**：
  - ✓ 确认 — 表示已知晓报警
  - 🔧 解决 — 标记报警已处理
- **分页**：支持翻页浏览
- **实时推送**：新报警通过 WebSocket 自动出现在列表顶部，导航栏铃铛图标显示红色角标

### 4. 系统配置

三组 Tab：

#### 设备管理
- 查看所有设备列表（编号、名称、品牌、车间、IP、启用状态）
- 添加设备：填写设备信息并关联协议模板
- 编辑 / 删除设备

#### 协议模板（核心配置）
- 查看已创建的 Modbus 协议模板列表
- 点击模板行展开查看所有寄存器项（名称、寄存器类型、地址、数据类型、系数、单位、是否计数器）
- 新建模板：填写模板元信息（名称、品牌、型号、版本），动态添加寄存器项
- 编辑模板：修改模板信息和寄存器项
- 删除模板

> 协议模板定义了设备"读哪些寄存器、如何解析"。接入新品牌机床的第一步就是创建对应的协议模板。

#### 用户管理（仅管理员）
- 查看用户列表
- 添加用户（用户名、密码、角色）
- 删除用户（不可删除 admin）

## 单机详情页

从车间总览点击设备行进入：

| 区域 | 说明 |
|------|------|
| 实时指标卡片 | 所有寄存器当前值（5s 刷新），点击卡片可查看该指标的时序趋势图 |
| 时序图表 | 选中指标后显示近 1/3/7 天的数值变化曲线 |
| 产量柱状图 | 近 7/14/30 天日产量统计 |
| 稼动率统计 | 最新一天的稼动率、运行时长、故障时长 |
| 报警历史 | 该设备的最近 20 条报警记录 |

## 常见操作流程

### 接入新设备

1. 系统配置 → 协议模板 → 新建模板，按设备文档配置寄存器
2. 系统配置 → 设备管理 → 添加设备，填写 IP/端口，选择模板
3. 返回车间总览，观察设备上线状态
4. 点击设备行查看实时采集数据

### 处理报警

1. 报警记录 Tab 或导航栏铃铛角标提示
2. 查看报警内容，判断原因
3. 点击 ✓ 确认已知晓
4. 现场处理后点击 🔧 标记已解决

### 导出数据

1. 统计报表 Tab
2. 选择日期范围
3. 点击「导出 CSV」下载文件
4. 用 Excel 打开进行二次分析
